# 🎓 MANTRA Academy Chennai Website

A responsive frontend website developed for the **MANTRA 2026 Summer School** assignment. The website represents an educational institute and provides information about the academy, courses, faculty, admissions, gallery, and contact details.

## 📌 Project Overview

MANTRA Academy Chennai is a modern educational institute website created using HTML, CSS, and JavaScript. The project focuses on responsive web design, user-friendly navigation, basic JavaScript interactivity, and SEO optimization.

> **Note:** This project is a **frontend-only website**. No backend or database has been implemented. Forms are validated using JavaScript only and do not store any user information.
